CSV API Documentation

Get All Data

Endpoint: GET /data.json

Example:

fetch('https://[username].github.io/[repository-name]/data.json')
    .then(response => response.json())
    .then(data => console.log(data));

Get Data by ID

Endpoint: GET /data_{id}.json

Example:

fetch('https://[username].github.io/[repository-name]/data_123.json')
    .then(response => response.json())
    .then(data => console.log(data));

Response Format

{
    "status": 200,
    "data": [
        // ข้อมูลจาก CSV
    ]
}
